# Task 3 – SQL Queries (Pagila Database)

This project contains solutions to SQL analytical tasks using the Pagila (DVD Rental) demo database.

## Tasks Covered
1. Films count per category
2. Top 10 most rented actors
3. Most profitable film category
4. Films not in inventory (without IN)
5. Top 3 actors in "Children" category (including ties)
6. Active vs inactive customers per city
7. Category with highest rental hours (filtered cities)

## Database
Pagila (PostgreSQL version of DVD Rental)
